# Máquina de Café com Poo

## Descrição

Este projeto consiste em uma simulação de uma máquina de café desenvolvida em Java utilizando Programação Orientada a Objetos.

O sistema permite ao usuário selecionar um dos seis sabores disponíveis através de uma interface gráfica utilizando JOptionPane.

Após a seleção, o sistema informa o sabor escolhido e seu respectivo valor.
